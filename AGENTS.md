## Project Overview
This is a Next.js + Supabase app for generating ecommerce images and video using Gemini for ecommerce startups.

## Tech Stack
- Next.js
- Supabase (Auth + DB)
- Tailwind CSS + Shadcn

## Commands
- dev: npm run dev
- build: npm run build

## Output Expectations
- Prefer minimal, working code
- Avoid over-engineering
